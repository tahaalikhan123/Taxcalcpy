# Taxcalcpy
Ontario Tax Calculator
This is a simple tax calculator application built using Python and the Tkinter library. It allows users to calculate their monthly salary after deducting federal and provincial taxes, as well as various deductions like RRSP, CPP, EI, and QPP.

How to Use
Run the Application: Execute the script to launch the tax calculator application.

Input User Details:

Enter your first name, last name, and phone number in the respective input fields.
Provide your monthly salary before taxes.
Input the amounts for RRSP, CPP, EI, and QPP deductions.
Calculate: Click the "Calculate" button to perform the tax calculations.

View Results: The application will display the following details:

First Name, Last Name, and Phone Number
Monthly Salary Before Taxes
Federal Tax
Provincial Tax
RRSP Deduction
CPP Deduction
EI Deduction
QPP Deduction
Monthly Salary After Taxes
Error Handling
The application performs input validation to ensure that all numerical inputs are valid and non-negative. It will display an error message if any input is invalid.
Customization
You can modify the tax calculation logic as needed. The current implementation uses simplified tax rates.
Requirements
Python 3.x
Tkinter library (usually included with Python)
